How to get a Gemini API key
  1. Open Google AI Studio
  2. Sign in with your Google account
  3. Click Create API key
  4. Copy the key (starts with AIza)
  5. Paste it above and hit Save

Free tier is enough for personal use. The key stays in this browser only — CalmFeed has no servers. Text and compressed media go straight to Google for embeddings.

Your key never leaves this browser except when calling Gemini. Uninstall removes local storage.
